# Changelog

## 0.2.0
- **Fixed toolbar button size**: Buttons now use fixed dimensions instead of flexible sizing
- **Configurable icon size**: Added `multiModuleFlutter.toolbarButtonSize` setting (40-150px, default 75px)
- **Improved UI consistency**: Better visual alignment and spacing of toolbar buttons across all sections
- **Settings integration**: Button size updates dynamically when settings change

## 0.1.0
- Initial release.
- Multi-module Flutter and Dart commands.
- Dashboard with grouped actions and quick commands.
- Module discovery via pubspec.yaml.
- Flutter logo folder icon.
